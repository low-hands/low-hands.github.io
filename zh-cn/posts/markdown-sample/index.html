    块引用 (Blockquotes)

    块引用（blockquote）元素代表引用自另一个来源的内容。它可以包含出处引用（必须包含在 footer 或 cite元素内），也可以包含行内修改（如注释和缩写）。

    无作者署名的块引用

    Tiam, ad mint andaepu dandae nostion secatur sequo quae. Note that you can use Markdown syntax within a blockquote.

    有作者署名的块引用

    — Rob Pike1
